北中南越文化差異 (Infographic 03)
====================================
北越（河內 / 海防）：性格較保守嚴謹 / 飲食偏清淡重魚露 / 重視傳統禮節與階級 / Tet 必備桃花（Hoa Dao）/ 四季分明冬季較冷 / 河內為千年文化古都 / 說話較直接但講究 / 對外國人較內斂

中越（順化 / 峴港）：性格溫和勤儉樸實 / 飲食偏辣重香料 / 順化宮廷菜最講究 / Tet 擺放香蕉與水果盤 / 歷史遺跡最多（會安/順化）/ 受占婆文化影響深 / 人民較為節儉 / 說話語調柔和緩慢

南越（胡志明市 / 芹苴）：性格開放熱情直接 / 飲食偏甜多樣化 / 受西方文化影響較深 / Tet 必備黃梅（Hoa Mai）/ 全年高溫僅雨旱兩季 / 經濟中心商業氛圍濃 / 對外國人較友善開放 / 生活節奏較快

文化融合小提醒：
- 越南 54 個民族各有獨特文化，以上為京族（Kinh）為主體的北中南概括
- 南北越因歷史發展路徑不同（社會主義 vs 資本主義），商業文化差異明顯
- 中越因歷史與地理因素，保留最多傳統宮廷文化與手工藝
- 與越南人互動時，先了解對方來自哪個區域，有助於避免文化誤解
