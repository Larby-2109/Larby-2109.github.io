## 本文重點摘要

- **台灣護照免簽入境富國島，最長停留30天，省下約TWD 600–900電子簽證費用**：唯一條件是必須從富國島國際機場（PQC）直接入境，不可經越南本土城市轉機再飛富國島（免簽僅限直飛或從第三國直飛PQC）。
- **直飛僅需3.5小時，越捷、星宇、中華、長榮四大航空皆有航班**：台灣每週超過20班直飛富國島，淡季來回票價可壓在TWD 4,500–6,000，是東南亞海島中性價比最高的直飛選擇之一。
- **5–10月雨季是體驗五星級度假村的黃金窗口**：雨季多為午後短暫陣雨（1–2小時即停），Regent、JW Marriott等頂級度假村常有5–6折促銷，每晚TWD 3,000–5,000即可入住附私人泳池的Villa。
- **島內交通以「租摩托車＋特定包車日」為最佳組合**：摩托車每日TWD 180–250，適合自由探索；跨南北島長距離移動（南島→北島約50–70分鐘車程）建議當日包車，單日約TWD 1,000–1,500。
- **南島玩海灘度假、中部吃夜市海鮮、北島玩主題樂園，三者各有主場**：建議南島住2晚（沙灘＋纜車）、北島住1–2晚（樂園＋動物園），中部作為過渡與美食據點，行程不走回頭路。
- **富國島整體物價約為台灣2/3、普吉島1/2**：背包客4天3夜可壓在TWD 7,000–11,000；舒適型約TWD 12,000–20,000；奢華Villa全包式也僅TWD 21,000–38,000／人。

富國島（Phú Quốc）是越南最大的島嶼，面積約574平方公里——大約是兩個台北市的大小。這座位於泰國灣（Vịnh Thái Lan）的熱帶海島，近年來以「免簽證＋直飛便利＋五星度假村高CP值」三大優勢，成為台灣旅客討論度最高的東南亞新興海島目的地。

不同於峴港（Đà Nẵng）的「城市＋海灘」混搭風格、也不同於芽莊（Nha Trang）的觀光團客感，富國島用一種相對原始卻極具現代度假品質的節奏，吸引著想逃離城市、沉浸在碧海藍天與白沙灘中的旅人。2025年太陽集團（Sun Group）持續加碼投資南島「日落小鎮」（Sunset Town）與跨海纜車周邊，2026年更有數間國際品牌度假村開幕，富國島的旅遊基礎建設正處於高速成熟期——現在去，正好。

## 一、為什麼選擇富國島？免簽證＋高CP值的越南海島天堂

### 1.1 台灣護照免簽30天：最直接的入境優勢

富國島是越南全境唯一對台灣護照實施「**免簽證入境30天**」的地區。只要你是從富國島國際機場（PQC）直接入境——無論是從台灣直飛、或從第三國（如韓國、泰國、新加坡）直飛富國島——即可免簽停留最多30天。

> ⚠️ **重要提醒**：免簽政策僅適用於**直接入境富國島國際機場**。如果你計畫先飛胡志明市（TP. Hồ Chí Minh）或河內（Hà Nội），再轉越南國內線飛富國島，則必須另外辦理越南電子簽證（eVISA，費用約USD 25 = TWD 800）。免簽只認「抵達PQC機場」這個點，不認最終目的地。

與越南本土城市相比，省下的不只簽證費用（TWD 600–900），還有排隊等待落地簽證（VOA）的時間成本。這對只有3–5天假期的短線旅人來說，是非常實在的優勢。

## 二、富國島最佳旅遊季節與天氣

### 2.1 乾季 vs 雨季：兩季體驗大不同

富國島屬於熱帶季風氣候，全年高溫（均溫26–29°C），沒有明顯的冬季。真正的差異在於降雨量與濕度，分為乾季（11–3月）與雨季（5–10月），4月與10月為過渡期。

| 比較項目 | 乾季（11–3月） | 雨季（5–10月） |
|---------|---------------|---------------|
| 降雨 | 每月5–8天，多為短暫陣雨 | 每月15–22天，午後雷陣雨為主 |
| 平均氣溫 | 22–31°C | 25–33°C |
| 濕度 | 65–75%（舒適） | 80–90%（潮濕悶熱） |
| 海水能見度 | 高（適合浮潛、潛水） | 中低（雨季浪大） |
| 住宿價格 | 旺季原價 | 5–6折促銷 |
| 觀光人潮 | 人多（尤其12–2月聖誕新年） | 人少、清幽 |
| 適合活動 | 沙灘、浮潛、跳島、纜車 | SPA、美食、樂園、度假村放空 |

### 2.3 淡旺季住宿價差

11–3月乾季為富國島旺季，尤其12月中至2月（聖誕新年＋農曆春節）為價格巔峰。五星級度假村如Regent Phu Quoc，旺季每晚TWD 10,000–15,000，淡季可能降至TWD 6,000–8,000。四星級度假村如Premier Residences，旺季TWD 3,000–4,500，淡季可壓至TWD 1,800–2,800。**5–10月是體驗頂級住宿的黃金窗口，價差可達2倍以上**。

## 三、富國島三大區域景點完整攻略

### 3.1 南島（Nam Đảo Phú Quốc）—— 沙灘、纜車、日落小鎮

南島是富國島最精華的景點集中區域，以Sunset Town為核心，涵蓋全島最著名的跨海纜車、Kiss Bridge（吻橋）及頂級沙灘。

#### 太陽世界跨海纜車（Cáp treo Hòn Thơm / Sun World Hon Thom Cable Car）

全長7,899.9公尺，曾創下金氏世界紀錄「最長跨海纜車」。從南島An Thới出發，跨越碧綠海域抵達Hon Thom（香島），單程約15分鐘。纜車視野極佳，可俯瞰漁船、小島與珊瑚礁——建議傍晚搭乘，夕陽景色為全島最美的觀景體驗之一。

| 項目 | 詳情 |
|------|------|
| 開放時間 | 08:30–17:00（最後入場16:00） |
| 門票 | 成人VND 650,000（約TWD 830）、兒童VND 450,000（約TWD 580） |
| 交通 | 南島包車或摩托車前往An Thới纜車站 |
| 建議停留 | 含纜車來回＋Hon Thom島上活動，約3–4小時 |

#### Kiss Bridge（Cầu Hôn／吻橋）

2024年底開放的南島新地標，由兩段不連接的橋面從海岸延伸向海中央，中間留有約30公分的間隙——設計概念為「親吻的距離」。日落時分，夕陽光線穿過橋面間隙的畫面，已成為富國島最經典的IG打卡畫面。

| 項目 | 詳情 |
|------|------|
| 開放時間 | 24小時開放（夜間有燈光） |
| 門票 | 免費（位於Sunset Town公共區域） |
| 交通 | Sunset Town步行可達 |
| 建議停留 | 傍晚1–2小時（拍日落＋夜間燈光） |

#### Bai Sao（Bãi Sao／星灘）

富國島最著名的白沙灘之一，沙質細白如粉、海水呈現翡翠綠漸層。海灘坡度平緩，浪小且安全，適合游泳與水上活動（水上摩托車、獨木舟、SUP立槳）。海灘周邊有數間海鮮餐廳與躺椅租借服務。

| 項目 | 詳情 |
|------|------|
| 開放時間 | 全天（水上活動約08:00–17:00） |
| 門票 | 免費（躺椅租借VND 50,000–100,000，約TWD 65–130） |
| 交通 | 南島度假村區車程約25–35分鐘 |
| 建議停留 | 半日（3–4小時，含午餐） |

#### Bai Kem（Bãi Kèm）

JW Marriott與Regent所在的頂級沙灘區域，沙質為全島最細、水色最清澈。海灘大部分為度假村專屬服務區，但依越南法律海灘皆為公有，非住客仍可進入。若預算有限無法入住，可安排至JW Marriott享用下午茶順便使用沙灘。

#### JW Marriott Phu Quoc Emerald Bay

由傳奇建築師Bill Bensley操刀，以「廢棄的法國大學」為主題的奇幻度假村。大廳是「圖書館」、餐廳是「化學實驗室」、SPA是「解剖學教室」——每個角落都充滿戲劇性的視覺驚喜。非住客可預約下午茶（約TWD 800–1,200／人）入內參觀拍照。

| 下午茶費用 | 約VND 650,000–950,000（約TWD 830–1,200） |
| 交通 | 南島包車約15–25分鐘 |
| 建議停留 | 2–3小時（拍照＋下午茶＋沙灘） |

### 3.2 中部（Trung Tâm）—— 夜市、文化、在地生活

中部以陽東鎮（Dương Đông）為核心，是富國島最早開發的區域，也是全島的行政、商業與美食中心。

#### 陽東夜市（Chợ Đêm Dương Đông / Dinh Cau Night Market）

富國島最大、最熱鬧的夜市，每日17:00–23:00營業。超過200個攤位涵蓋海鮮大排檔、越南小吃、水果冰沙、紀念品與服飾。是品嚐富國島海鮮最方便也最經濟的選擇——一盤現烤龍蝦約TWD 300–500，烤大頭蝦約TWD 150–250。

| 項目 | 詳情 |
|------|------|
| 營業時間 | 每日17:00–23:00 |
| 人均消費 | TWD 200–500（不含高價海鮮） |
| 交通 | 中部步行可達；南島包車約30分鐘 |
| 建議停留 | 1.5–2小時 |

#### Dinh Cậu（小龍王廟）

位於陽東鎮海濱岩石上的小廟，是富國島漁民的信仰中心，供奉海上守護神「小龍王」。廟宇規模不大，但黃昏時分的海景視野極佳，是觀賞富國島經典日落的地點之一。

#### 魚露工廠（Nhà Thùng Nước Mắm）

富國島是越南魚露（Nước Mắm）的核心產區，全盛時期島上有超過100座魚露發酵桶倉。參觀魚露工廠可了解這項越南飲食靈魂的製作過程——以小魚（多為鯷魚）與鹽分層堆疊、在巨大木桶中發酵12–18個月。多數工廠免費參觀，現場可購買不同年份與等級的魚露（價格從VND 50,000至VND 500,000+）。

#### 胡椒園（Vườn Tiêu Phú Quốc）

富國島的胡椒（Hồ Tiêu）以香氣濃郁、辣度溫和聞名，是越南重要的胡椒產區。島上有多座胡椒園開放參觀，可近距離觀察胡椒藤蔓的生長狀態，了解黑胡椒、白胡椒與紅胡椒的加工差異。現場通常提供胡椒試吃與伴手禮購買。

### 3.3 北島（Bắc Đảo Phú Quốc）—— 樂園、動物園、原始海岸

北島是富國島開發最晚的區域，保留了大量的原生態森林與海岸線。這個區域的旅遊主力為Vinpearl（珍珠集團）旗下的主題樂園與野生動物園。

#### VinWonders Phú Quốc（珍珠奇幻樂園）

越南最具規模的主題樂園，佔地約50公頃，分為六大主題區：歐洲街、童話世界、冒險世界、水上樂園、颱風世界與海龍世界。水上樂園為全越南最大，設有造浪池、漂漂河與多條高速滑水道，足以讓親子家庭玩上整整一天。

| 項目 | 詳情 |

## 四、富國島交通全攻略

### 4.1 島外交通：台灣 ↔ 富國島

從台灣前往富國島，主要有直飛與轉機兩種方式。

**直飛**（首選）：越捷、星宇、中華、長榮四家航空提供台灣直飛富國島國際機場（PQC）的航班。飛行時間約3小時20分–3小時30分，是前往富國島最便捷的方式。直飛航班主要從桃園國際機場（TPE）出發，越捷另有高雄（KHH）出發的航線。

**轉機**：可經胡志明市（SGN）或河內（HAN）轉越南國內線至富國島。總航程約6–8小時（含轉機等待）。轉機選項需注意：若你計畫使用富國島免簽政策，**必須確保你的國際航班是直飛PQC**——若先飛胡志明市再轉機，則需要辦理越南簽證。

### 4.2 島內交通方式

富國島全長約48公里（南北向），島內交通選擇如下：

| 交通方式 | 每日費用（TWD） | 優點 | 缺點 | 適合族群 |
|---------|---------------|------|------|---------|
| 租摩托車 | 180–250 | 自由度最高、費用最低 | 需國際駕照、雨季不便、路況不熟有風險 | 有騎車經驗的年輕旅人 |
| Grab叫車 | 單趟100–350 | 方便、價格透明、免議價 | 偏遠區域（北島、南島部分）車輛少 | 短程移動、中部區域 |
| 包車一日 | 1,000–1,800 | 舒適、有冷氣、司機可兼導遊 | 費用較高 | 家庭、長距離跨區移動 |
| 計程車 | 單趟150–500 | 隨時攔車、方便 | 需議價或確認跳錶 | 機場接送、短程 |
| 飯店接駁 | 多數免費（定時定點） | 免費、方便 | 班次有限、路線固定 | 度假村住客 |

### 4.3 交通組合建議

- **南島旅客**：租摩托車為主力（南島路況較好、車流較少），搭配1–2天包車去北島與中部。
- **中部旅客**：Grab＋步行即可覆蓋大部分需求。
- **北島旅客**：Vinpearl度假村住客多數有免費接駁車往返樂園與機場；若需外出，建議包車或叫Grab。
- **跨南北島**：單程約50–70分鐘車程，包車單趟約TWD 500–900，建議將南北島行程分開安排（前幾晚住南島、後幾晚住北島），減少不必要的長途移動。

### 4.4 租摩托車注意事項

富國島租摩托車價格為東南亞最實惠的等級之一——每日VND 120,000–180,000（約TWD 150–230）。租車時需注意：

1. **國際駕照**：富國島部分租車行要求出示國際駕照＋本國駕照，建議出發前至監理所辦理（TWD 250）。
2. **安全帽**：越南法規強制佩戴安全帽，租車時通常會附贈，未戴罰款VND 200,000–300,000。
3. **押金**：多數車行不收押金，改為押護照或護照影本——建議提供影本而非正本。
4. **加油**：島上加油站充足，92無鉛汽油約VND 20,000–23,000／公升，加滿一桶約TWD 80–120。

## 九、常見問題（FAQ）

**Q1：台灣護照去富國島真的免簽嗎？條件是什麼？**

是的。持有台灣護照的旅客，只要從富國島國際機場（PQC）直接入境（無論從台灣直飛或從第三國直飛），即可免簽停留最多30天。若你先飛越南本土城市（胡志明市、河內）再轉機到富國島，則需辦理越南電子簽證（eVISA）。

**Q2：富國島用什麼貨幣？需要先換越南盾嗎？**

越南盾（VND）為主要流通貨幣。富國島機場、陽東鎮有ATM與換匯所。台灣出發可先換少量越南盾（約TWD 3,000–5,000）作為機場到飯店交通費，其餘到當地再用ATM提領（匯率通常較佳）。部分度假村與大餐廳接受Visa/Mastercard信用卡。

**Q3：富國島講什麼語言？英文通嗎？**

官方語言為越南語。富國島的國際度假村（JW Marriott、Regent、New World等）前檯英文流利；夜市、街邊攤販與摩托車租車行英文程度有限，建議準備Google翻譯App或重要地點的越南文名稱截圖。

**Q4：富國島網路與SIM卡怎麼買？**

富國島機場入境大廳有多間電信櫃檯（Viettel、Mobifone、Vietnamobile），旅遊SIM卡（7–15天上網不限流量）約VND 100,000–200,000（TWD 130–260）。台灣出發前也可在KKday、Klook購買越南eSIM，提前安裝、落地即用。詳細推薦請參閱[越南SIM卡與eSIM推薦（2026最新版）](/posts/guides/vietnam-sim-card-esim-guide-2026/)。

**Q5：富國島安全嗎？有什麼需要注意的？**

富國島整體治安良好，是越南相對安全的旅遊地區。需注意：夜市人多時注意扒手、不要將貴重物品留在摩托車座墊下、租摩托車時建議提供護照影本而非正本作為押金。海灘游泳注意安全旗標（紅旗表示禁止下水）。

**Q6：富國島要給小費嗎？**

越南沒有強制小費文化，但觀光區已逐漸接受小費。建議：餐廳若有服務費（5%–10%）可不另給；按摩師傅可給VND 50,000–100,000（TWD 65–130）；包車司機全日服務可給VND 100,000–200,000（TWD 130–260）。

**Q7：富國島插頭與電壓？需要帶轉接頭嗎？**

電壓220V、50Hz，插座多為兩腳扁型（與台灣相同）或兩腳圓型。台灣的兩腳扁型插頭可直接使用，不需轉接頭。三腳插頭（如某些筆電充電器）可能需要轉接頭。

**Q8：雨季去富國島會不會很掃興？**

不一定。雨季（5–10月）的降雨多為午後短暫陣雨（通常1–2小時即停），上午與傍晚仍可正常活動。雨季的優勢是住宿價格大幅下降（5–6折）、遊客極少、沙灘清幽。如果你不介意午後可能會下一場雨，雨季反而是體驗頂級度假村的最划算時機。

**Q9：富國島可以租汽車自駕嗎？**

不建議。越南不承認台灣國際駕照（台灣非《日內瓦道路交通公約》締約國），無法合法租汽車自駕。但摩托車租車行多數接受台灣國際駕照（或甚至僅看本國駕照），實際執行較寬鬆。安全起見，建議以包車或Grab作為主要交通方式。

**Q10：富國島適合帶長輩或小孩去嗎？**

非常適合。富國島的度假村設施完善、沙灘坡度平緩安全、主題樂園與動物園適合全年齡層。建議帶長輩時選擇包車而非摩托車作為島內交通，住宿選擇南島或北島的大型度假村（醫療設施較完善、餐廳選擇多元）。帶小孩的首選是北島Vinpearl系列（全包方案含樂園門票＋三餐，最省心）。

**Q11：富國島可以跳島玩嗎？附近有哪些小島？**

可以。富國島南部An Thới群島（包括Hon Thom香島、Hon Móng Tay指甲島、Hon Gầm Ghì等）提供跳島浮潛一日遊行程，可在南島旅行社或Klook/KKday預訂，含飯店接送、午餐與浮潛裝備，每人約TWD 800–1,500。最佳跳島季節為乾季（11–3月），雨季海浪較大、部分航線可能取消。

**Q12：富國島有便利商店或超市嗎？**

陽東鎮有VinMart（Vin集團旗下超市）與數間小型便利商店，可購買飲料、零食、防曬乳與基本生活用品。南島與北島的度假村周邊便利商店較少，建議入住前先在陽東鎮的超市補充所需物資。
